Class: DbEntityAnchorMapper

DbEntityAnchorMapper()

Used to create arrays of "anchor objects" from an array of schema objects (from a FilterableSchemaList instance). These anchor objects are used to create anchor tags that take you to the individual entity records on a page. Each object will have these properties:

"anchor" (the HTML element ID for the entity); "label" (anchor tag text for the entity); "isMatch" (was this entity a match to a filter?); "entityID" (the ID in the entity object); "entityType" (the name of the entity type -- "schema"/"table"/"column")
